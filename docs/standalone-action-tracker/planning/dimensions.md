# Dimensions Selection

The dimension selection area allows you to select the organisation unit and period you want to perform planning for.
These inputs use standard DHIS2 selectors for both organisation unit and period. If you had set the default period and
organisation unit in the configuration, those values will be pre-selected whenever you open the application

## Selecting an organisation unit

To select or update the organisation unit value, click on the gray area under the label `Select organisation unit`, a
popup with organisation unit list will appear. Select the organisation unit you want and click on the update. The
application will load planning data for the selected organisation unit.

![org_unit_selection](/img/standalone-action-tracker/org_unit_selection.png)

:::info
Your planning organisation unit selection will be limited to the planning organisation unit level specified in the
[configuration](../configuration/general#planning-organisation-unit-level).
:::

## Selecting a period

To select or update the period, click on the gray area under the label `Select Period`, a popup with applicable period
types and periods will appear. Select the period and click on update. The application will load planning data for the
selected period.

![period_selection](/img/standalone-action-tracker/period_selection.png)

:::info
Your planning period selection will be limited to the period type specified in the
[configuration](../configuration/general#planning-period).
:::
